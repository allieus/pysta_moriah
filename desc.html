실습 2: 로그인 / 아웃

 

step 1. 로그인/아웃을 체크할 수 있는 페이지를 만든다

- /member/is_login 이라는 주소로 로그인 체크를 할 수 있는 뷰를 만들고

- urls.py 에 연결한다.

- 해당 뷰는 로그인/아웃 여부를 화면에 출력한다

/pysta/urls.py

/member/views.py

 

step 2. urls.py 에 로그인 뷰를 연결한다

- /member/login 주소로 한다

/pysta/urls.py

 

step 3. 로그인 폼을 작성한다

/templates/member/login.html

 

step 4. 로그인 처리를 한다

- member 앱의 login 이라는 이름의 view 로 처리한다

/member/views.py

 

step 5. urls.py 에 로그아웃 주소를 /member/logout 으로 연결한다

/pysta/urls.py

 

step 6. 확인한다

- /member/is_login 호출하여 로그아웃 상태 확인

- /member/login 호출하여 로그인 하기

- /member/is_login 호출하여 로그인 상태 확인

- /member/logout 호출하여 로그아웃 하기

- /member/is_login 호출하여 로그아웃 상태 확인
