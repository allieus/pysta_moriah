실습 10: 사진 업로드

 

step 1. 사진 업로드 될 모델 만들기

- timeline 이라는 앱을 하나 만듭니다

- timeline/models.py 를 열고 모델 Post 를 만듭니다

- Post 에는 다음과 같은 필드를 설정합니다

- 글 내용이 들어갈 contents

- 이미지 파일명이 들어갈 image

- 작성자가 들어갈 owner

- 작성일시가 들어갈 created_at

/timeline/models.py

 

step 2. urls.py 에 프로필 페이지 주소를 삽입하기

- member view 의 profile 이라는 뷰를 "/profile" 이라는 주소로 연결합니다

/pysta/urls.py

 

step 3. 프로필 페이지 탬플릿 만들기.

- 이미지 파일과, 간단한 설명을 입력하는 form 을 /template/member/profile.html 이라는 파일로 작성합니다

/templates/member/profile.html

 

step 4. profile view 만들고 업로드 하기

- 회원 정보에서 했던 것 처럼, 업로드를 처리합니다.

- 이번 썸네일 크기는 600x900 으로 합니다.

- 업로드 위치는 /media/post, /media/post/thumbnail 입니다

/member/views.py

 

step 5. Post 모델에 저장하기

- 업로드 된 이미지, 받아온 텍스트, 글쓴이 정보를 Post 모델로 추가합니다

 

step 6. 확인하기

- DB 의 timeline_post의 값을 확인합니다

- 이미지가 잘 올라갔는지 확인합니다.
