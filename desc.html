실습 3: 가입 인증

 

step 1. 가입시 자동 인증을 하지 않도록 바꾸기

- 회원 가입 모델에서, 사용자를 생성 후, 해당 사용자의 active 상태를 False 로 바꾸기

/member/views.py

 

step 2. 로그인 시 active 상태를 체크하게 바꾸기

- 회원 로그인 페이지에서, 회원의 active 상태를 확인하고, active 한 경우에만 로그인 시키기

 

step 3. 인증 model 만들기

- member/models.py 에 ActivationKey 모델을 생성

- user 라는 field 로 사용자 모델을 PK 로 연결

- activation_key 라는 field 로 64자리 텍스트 필드를 유니크하게 생성

/member/models.py

 

step 4. 회원 가입 시 인증 키 생성

- 랜덤한 64자 인증 키 생성

- ActivationKey 모델에 해당 키가 존재하는지 검색하고

- 없는 경우 키와 이메일로 새 object 를 생성

/member/views.py

 

step 5. 회원 가입 시 생성된 인증 키를 메일로 발송

- html 로 메일 보내기

- 다음의 주소를 링크하여 보내기

- "http://서비스 주소/member/activate?key=생성된 키"

 

step 6. activate view 만들기

- member/views.py 에 activate view 만들기

- ActivationKey 모델에서 키 검색하기

- 찾은 데이터에 해당하는 사용자의 active 상태를 True 로 변경하기

- urls.py 에 등록하기

/pysta/urls.py
