실습 4: 완성도 높이기

 

step 1. 메일 발송 시 template 이용하기

- 탬플릿 폴더에 mail_form 폴더를 만들고 activate.html 파일을 생성

- 인증 메일을 보낼 때 사용한 html 내용을 입력

- 렌더링 한 결과를 html_message로 보내기

/templates/mail_form/activate.html

 

step 2. 로그인 시 인증이 안된 사용자는 재인증 페이지로 보내기

- 재인증 view 를 만들고 /member/issue_activation 주소로 연결

- 해당 view 에서는 이메일을 입력받고, 기 인증 여부를 확인

- 인증되지 않은 사용자라면, 회원 가입 시 했던 것과 같은 방법으로 메일을 발송

- 기존 존재하는 키를 단순히 다시 발송해도 되고, 새 키를 만들어도 됨.

/member/views.py

/templates/member/issue_activation.html

 

step 3. 회원 가입 시 에러 체크

- 회원 가입 시 패스워드 동일 여부, 중복된 이메일 / username 체크

- 메일 형식 체크

/member/views.py

/templates/member/register.html
